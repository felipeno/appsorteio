# appsorteio
Sorteio APP Java com AndroidSDK

Para utilizar o aplicativo basta clonar o repositório em sua máquina e executá-lo utilizando
AndroidSDK.

A ideia do aplicativo é sortear um número que esteja entre o intervalo de números que o usuário
disponibiliza.

Observações:
i. Não será sorteado nenhum valor caso haja campos vazios;
ii. Não será sorteado nenhum valor caso o valor mínimo (primeiro campo) seja maior que o valor
máximo (segundo campo).
